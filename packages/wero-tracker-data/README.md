# Data for the Wero Adoption Tracker

This repository contains all the data for the [Wero Adoption Tracker](https://werotracker.eu) ([GitHub](https://github.com/sharknoon/wero-tracker)).

## How to contribute

This project heavily relies on up-to-date informations about the current state of the Wero Adoption of individual banks.

If you want to help this project by reviewing, changeing or adding data, please fork this repository and make a Pull Request.

## Guides

### How to check if a Bank is available on the Wero App (instead of / in addition to their own banking app)

1. Download the official Wero App for [iOS](https://apps.apple.com/de/app/wero-wallet/id6480469576) or [Android](https://play.google.com/store/apps/details?id=eu.epicompany.wero.wallet)
2. Open the app
3. Select the bank in question
4. You now either see a message, that this bank is not supported or a login screen for the bank.
